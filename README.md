# notes-tak22

This is a simple notes app built with Vue and Express. It uses plain json array to store notes.

## Project management

[https://www.pivotaltracker.com/n/projects/2632252](https://www.pivotaltracker.com/n/projects/2632252)

## Project setup

```
cp .env.example .env
```

```
npm install
```

### Running the app

```
npm start
```
